# JobTest-BackEnd-1

## Navodila-SLO

Napiši CRUD API z uporabo Express.js za routing in Mongoose za interakcijo z MongoDB bazo.

Seznam poti, ki jih naj API omogoča:
**[GET] /products**
**[POST] /product**
**[GET] /product/:id**
**[DELETE] /product/:id**
**[PUT] /product**

    name : 'Computer 1',
